1
링크 접속 후 이름 + 숫자 4자리 비밀번호 입력
2
처음이면 자동으로 가입돼요 — 별도 회원가입 불필요
3
레벨 선택 (초급 / 중급 / 고급 / 심화 / 기존수강생) 후 시작일 입력

같은 이름 + 같은 비번으로 들어오면 폰을 바꿔도, 앱을 지웠다 깔아도 기록이 그대로 이어져요.
📅 캘린더 탭
30일 챌린지의 핵심 공간이에요
매일 체크하고, 기록하고, 챌린지방에 제출하는 모든 흐름이 여기서 완결돼요.
📊
진행 현황 배너
오늘이 몇 일차인지, 전체 30일 중 몇 % 진행됐는지 한눈에 보여요.
레벨과 이름도 함께 표시되고, 날짜가 틀렸으면 ✏️ 날짜 수정 버튼으로 언제든 고칠 수 있어요.
🗓️
30일 달력
오늘 날짜 칸을 누르면 완료 체크/해제가 돼요. 초록색으로 바뀌면 오늘 완료!
지난 날짜 칸을 누르면 그날의 투두와 결과를 수정할 수 있어요.
공구일정이 확정된 날엔 달력에 🛒 표시가 자동으로 생겨요.
📋
오늘의 투두리스트
오늘 할 일을 입력하고 하나씩 체크하면서 진행해요.
완료 못 한 투두는 다음 날 자동으로 이월돼요 — 놓친 게 있어도 걱정 없어요.
투두 항목은 추가/삭제 모두 자유롭게 할 수 있어요.
✨
오늘 해낸 결과 기록
수치, 건수, 구체적인 성과를 자유롭게 적어요.
예) DM 제안 5명 / 메일 제안 3명 / 제안 제품: 유기농 그래놀라
결과 저장 버튼 누르면 저장 완료!
📤
챌린지방 제출 양식 자동생성
✨ 양식 생성 버튼 누르면 투두 + 결과가 합쳐진 제출 양식이 자동으로 완성돼요.
📋 복사 버튼으로 바로 복사해서 챌린지방에 붙여넣기하면 끝!
레벨, 이름, 몇 일차인지가 자동으로 들어가요.
💡 이렇게 쓰면 가장 효과적이에요
하루 시작할 때 투두 입력 → 하루 끝날 때 결과 기록 → 양식 생성 후 챌린지방 제출
딱 이 흐름으로만 써도 30일이 체계적으로 쌓여요 😊
🛒 공구 탭
공구일정 관리 + 도구 모음
내 공구 진행 상황을 관리하고, 필요한 도구를 바로 열 수 있어요.
🛒
공구일정 관리
협찬 / 공구예정 / 공구일정 확정 3가지 유형으로 등록할 수 있어요.
셀러 인스타 주소, 상품명, 시작일~종료일, 메모까지 한 번에 기록해요.
공구일정 확정으로 등록하면 캘린더 달력에도 🛒 표시가 자동으로 생겨요.
입점 연락이 왔을 때 💬 메리에게 공유! 버튼으로 바로 공유할 수 있어요.
⚠️
제품이슈 탭 NEW
메리 + 수강생이 올린 제품 이슈를 전체 공유하는 공간이에요.
입점 제안 전에 꼭 확인하세요 — 물량 수급 문제, 발송 지연, 업체 이슈 등이 올라와요.
입점완료 제품에서 이슈가 생기면 입점탭 → 이슈 발생 신고로 등록하면 여기에 자동으로 공개돼요.
⚡ 바로가기 도구 3종
⚡ 5분자동화 📄 제안서(간편) 🃏 공구카드
⚡
5분자동화
입점제안 메일 + DM + 제안서까지 한 번에 자동으로 만들어주는 도구예요.
상품마스터를 통한 제안서 작성과 DM 양식까지 포함돼 있어서 조금 시간이 걸리지만 가장 완성도 높은 제안서가 나와요.
처음 쓰는 분은 도구 안의 ? 사용법 버튼을 먼저 눌러보세요!
📄
제안서 간편버전 업데이트
급할 때 빠르게 쓰기 좋은 간편 제안서예요.
쇼핑몰관리자 → 인플페이 → 수익시뮬레이터에서 해당 상품 불러와
인플루언서 마진 퍼센트 측정 후 가격 확인 → 입력 → PDF 또는 PNG 저장 → 전달 끝!
5분자동화와 상황에 맞게 둘 다 활용해요.
🃏
공구카드 메이커
인플루언서 공구할 때 필요한 공구카드를 직접 만들어서 바로 저장할 수 있어요.
리스트형 / 그리드형 / 테이블형 / 오픈카드 4가지 스타일 선택 가능
컬러·폰트·배경 패턴 선택, 옵션별 이미지·가격·배지 입력
인스타 4:5 비율 고정 기능 있어서 저장하면 PNG로 바로 인스타 업로드 가능!
💡 도구 선택 가이드
시간 여유 있을 때 → 5분자동화 (완성도 최고)
급하게 제안서만 필요할 때 → 제안서 간편버전
공구 카드 만들 때 → 공구카드 메이커
🏪 입점 탭
입점 현황 관리 + 소통방
내 입점 진행 상황을 단계별로 관리하고, 수강생끼리 정보를 나눠요.
✉️
입점제안 메일 생성기 NEW
입점 제안 메일을 3분이면 완성할 수 있어요.
제품 정보만 입력하면 메일이 자동으로 만들어지고, 카테고리별 실제 성과 수치가 자동으로 들어가요.
1
내 이름 · 직책 입력 (서명 자동 생성)
2
카테고리 선택 — 식품 / 유아동 / 생활 / 뷰티 / 패션 / 반려동물
3
제품명 입력 (띄어쓰기 포함 전체)
4
업체명 입력 (모르면 미정 클릭 → 메일에 '귀사'로 표시)
5
제품 특징 한 줄 입력 (선택, 없으면 건너뛰기)
6
제목 4가지 + 본문 4가지 중 골라서 복사 → 발송!
⚠️ 메일 보낼 때 아래 두 파일 꼭 첨부하세요!
📎 입점제안서 → 셀럽즈 벤더몰에서 다운로드
📎 판매 레퍼런스 → 공지사항 이용가이드/입점 글에서 다운로드
🏪
입점 현황 관리
입점 진행 상황을 6단계로 나눠서 관리해요.
📤 입점제안완료 → 📞 입점연락 옴 → ⚙️ 입점진행중 → ✅ 신청완료 → 🎉 입점완료

입점완료 된 제품에서 이슈가 생기면 ⚠️ 이슈 발생 신고 버튼으로 신고하면
공구탭 제품이슈에 자동으로 공개돼서 다른 수강생들도 알 수 있어요.

입점 연락이 왔을 때 💬 메리에게 공유! 버튼으로 바로 카카오톡 공유 가능!
🤝
입점소통방 NEW
입점 제안하다 거절당한 업체를 수강생끼리 공유하는 공간이에요.
한 명이 거절당한 업체, 다음 사람은 안 가도 되잖아요 🙏

제안 전에 꼭 검색 먼저! 이미 등록된 업체인지 확인하고 제안하세요.
거절당했으면 업체명 + 제품명 + 이유까지 등록해주면 다음 사람이 시간을 아낄 수 있어요.
등록자 이름은 첫 글자만 공개돼요 (익명 보호).
💡 입점탭 활용 순서
제안 전 → 입점소통방 검색 확인
메일 작성 → ✉️ 메일 생성기 활용
제안 후 → 입점현황에 단계별 등록
거절당하면 → 입점소통방에 등록해서 공유
📢 공지 탭
메리의 공지사항
중요한 공지는 여기서 확인하세요. 놓치면 손해인 내용들이 올라와요.
📌
상단 고정 공지
가장 중요한 공지는 📌 상단에 고정돼요.
항상 먼저 확인해주세요 — 교육 일정, 중요 안내 등이 올라와요.
🆕
NEW 표시
아직 읽지 않은 공지에는 NEW 뱃지가 붙어요.
탭 버튼에도 숫자가 표시돼서 얼마나 안 읽었는지 바로 알 수 있어요.
공지를 열면 자동으로 읽음 처리돼요.
🔔
새 공지 팝업
캘린더에 접속하면 읽지 않은 공지가 있을 때 자동으로 팝업이 떠요.
팝업에서 바로 전체 공지 보기로 이동할 수 있어요.
💡 공지는 꼭 읽어주세요!
교육 일정 변경, 새 기능 안내, 중요 공지가 여기 올라와요.
NEW 표시가 있으면 바로 확인하는 습관을 들여주세요 😊
📆 교육일정 탭

오프교육, 줌스터디, 피드백 일정을 월별로 확인할 수 있어요.

오프교육

신청은 공지사항 또는 메리 공지를 통해 별도로 진행돼요.
💜
줌스터디 / 줌교육
온라인 줌으로 진행되는 스터디 및 교육이에요.
링크는 당일 카카오 공지로 안내돼요.
💛
피드백만 진행
별도 교육 없이 1:1 또는 그룹 피드백만 진행되는 날이에요.
🩶
휴일
교육이 없는 날이에요. 긴급 피드백 외 교육 휴무예요.
